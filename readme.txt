Makes your WordPress admin dashboard in dark mode.

== Description ==

Adds a smooth dark mode to the default WordPress dashboard interface. The dark mode can be disabled per user in the profile page.

Plugins officially supported:
* Jetpack
* Nested Pages
* Yoast SEO
* WooCommerce

== Installation ==

1. Upload the plugin package to the plugins directory.
2. Activate the plugin.
3. Done! Your WordPress dashboard is now in dark mode :)
